Arrays

An array stores a fixed-size sequential collection of elements of the homogenous(same) type. An array is used to store a collection of data, but it is often more useful to think of an array as a collection of variables of the same type stored at contiguous memory locations.

Instead of declaring individual variables, such as number0, number1, ..., and number99, you declare one array variable such as numbers and use numbers[0], numbers[1], and ..., numbers[99] to represent individual variables. A specific element in an array is accessed by an index.

All arrays consist of contiguous memory locations. The lowest address corresponds to the first element and the highest address to the last element.

 

Declaring Arrays

Array Declaration Example

datatype[] arrayName;

where,

  • datatype is used to specify the type of elements in the array.
  • [ ] specifies the rank of the array. The rank specifies the size of the array.
  • arrayName specifies the name of the array.

Initializing an Array

double[] balance;

Declaring an array does not initialize the array in the memory. When the array variable is initialized, you can assign values to the array.

Array is a reference type, so you need to use the new keyword to create an instance of the array.

Create Instance of Array

double[] balance = new double[10];

Assigning Values to an Array

You can assign values to individual array elements, by using the corresponding index number

double[] balance = new double[10];

balance[0] = 4500.0;

You can assign values to the array at the time of declaration, as shown:

double[] balance = { 2340.0, 4523.69, 3421.0};

You can also create and initialize an array, as shown:

int [] marks = new int[5]  { 99,  98, 92, 97, 95};

You may also omit the size of the array, as shown:

int [] marks = new int[]  { 99,  98, 92, 97, 95};

You can copy an array variable into another target array variable. In such case, both the target and source point to the same memory location:

int [] marks = new int[]  { 99,  98, 92, 97, 95};

int[] score = marks;

When you create an array, C# compiler implicitly initializes each array element to a default value depending on the array type. For example, for an int array all elements are initialized to 0.

Accessing Array Elements

An element is accessed by indexing the array name. This is done by placing the index of the element within square brackets after the name of the array. For example,

double salary = balance[9];

Array Example


//This example demonstrates array declaration, assignmnet, and accessing arrays


using System;

namespace ArrayApplication

{

   class MyArray

   {

      static void Main(string[] args)

      {

         int []  n = new int[10]; /* n is an array of 10 integers */

         int i,j;

 

         /* initialize elements of array n */

         for ( i = 0; i < 10; i++ )

         {

            n[ i ] = i + 100;

         }

        

         /* output each array element's value */

         for (j = 0; j < 10; j++ )

         {

            Console.WriteLine("Element[{0}] = {1}", j, n[j]);

         }

         Console.ReadKey();

      }

   }

}

Element[1] = 101

Element[2] = 102

Element[3] = 103

Element[4] = 104

Element[5] = 105

Element[6] = 106

Element[7] = 107

Element[9] = 109

Using the Foreach loop

In the previous example, we used a for loop for accessing each array element. You can also use a foreach statement to iterate through an array.

using System;

namespace ArrayApplication

{

   class MyArray

   {

      static void Main(string[] args)

      {

         int []  n = new int[10]; /* n is an array of 10 integers */

        

         /* initialize elements of array n */

         for ( int i = 0; i < 10; i++ )

         {

            n[i] = i + 100;

         }

        

         /* output each array element's value */

         foreach (int j in n )

         {

            int i = j-100;

            Console.WriteLine("Element[{0}] = {1}", i, j);

           

         }

         Console.ReadKey();

      }

   }

}

Element[3] = 103

Element[4] = 104

Element[5] = 105

Element[6] = 106

Element[7] = 107

Element[9] = 109

Lists

A list is an object which holds variables in a specific order. The type of variable that the list can store is defined using the generic syntax.

List Declaration Example

List<int> numbers = new List<int>();

Array & List Differences

The difference between an array and a list is that lists are dynamically sized and arrays are fixed in size.

When you are uncertain about the size of the container required, just use a list.

List Add Example

List<int> numbers = new List<int>();
numbers.Add(1);
numbers.Add(2);
numbers.Add(3);

It's also possible to add an entire array to a list using the AddRange function.

Add Array To List

List<int> numbers = new List<int>();
int[] array = new int[] { 1, 2, 3 };
numbers.AddRange(array);

You can also remove contents from a list using Remove

Remove List Item

List<string> groceries = new List<string>();
// add groceries
groceries.Add("soda");
groceries.Add("cookies");
groceries.Add("ham");

// now remove the cookies
groceries.Remove("cookies");
Console.WriteLine(groceries.Count);

Output

2

We can also use the RemoveAt function to specifiy an index of an item to remove. In our case, to remove the cookies, we will use the index 1.

RemoveAt

List<string> groceries = new List<string>();
// add groceries
groceries.Add("soda");
groceries.Add("cookies");
groceries.Add("ham");

// now remove the cookies
groceries.RemoveAt(1);
Console.WriteLine(groceries.Count);

Output

2



Concatenating Lists

You can use the AddRange function to join lists

Concatenating Lists Example

List<string> food = new List<string>();
food.Add("steak");
food.Add("chicken");

List<string> vegetables = new List<string>();
vegetables.Add("tomato");
vegetables.Add("carrot");

food.AddRange(vegetables);
Console.WriteLine(food.Count);

Output

4
