C# Syntax Information

C#'s syntax is quite similar to both C++ and Java. If you've had any previous experience with either C++ or Java you shouldn't have too much difficulty learning C#'s syntax.
But the main point to remember is that In Object-Oriented Programming methodology, a program consists of various objects that interact with each other by means of actions.
The actions that an object may take are called methods. Objects of the same kind are said to have the same type or, are said to be in the same class.

For example, let us consider a Rectangle object. It has attributes such as length and width.
Depending upon the design, it may need ways for accepting the values of these attributes, calculating the area, and displaying details.


Rectangle Program


using System;
namespace RectangleApplication
{
class Rectangle
{
// member variables
double length;
double width;
public void Acceptdetails()
{
length = 4.5;
width = 3.5;
}

public double GetArea()
{
return length * width;
}
public void Display()
{
Console.WriteLine("Length: {0}", length);
Console.WriteLine("Width: {0}", width);
Console.WriteLine("Area: {0}", GetArea());
}
}

class ExecuteRectangle
{
static void Main(string[] args)
{
Rectangle r = new Rectangle();
r.Acceptdetails();
r.Display();
Console.ReadLine();
}
}
}

Output

Length: 4.5
Width: 3.5
Area: 15.75

I will now begin explaining some definitions and what some of these lines of code mean

Using Keyword

The first statement in a C# program is usually..

using System;

The using keyword is used for including the namespaces in the program. A program can include multiple using statements.

Semicolons

Semicolons are used to terminate statements, for instance this signifies the end of this expression.

length = 4.5;

This simply assigns the value 4.5 to the variable length

The Class Keyword

The class keyword is used for declaring a class

Identifiers

An identifier is a name used to identify a class, variable, function, or any other user-defined item. The basic rules for naming classes in C# are as follows:

  • It should not be a C# keyword.
  • A name must begin with a letter that could be followed by a sequence of letters, digits (0 - 9) or underscore. The first character in an identifier cannot be a digit
  • It cannot contain any embedded space or symbol such as ? - + ! @ # % ^ & * ( ) [ ] { } . ; : " ' / and \. However, an underscore ( _ ) can be used.

Member Variables

Variables are attributes or data members of a class, used for storing data. In the rectangle program, the Rectangle class has two member variables named lengthwidth .

Member Functions

Functions are set of statements that perform a specific task. The member functions of a class are declared within the class.
Our sample class Rectangle contains three member functions: AcceptDetails, GetArea and Display .

Instantiating a Class

In the rectangle program, the class ExecuteRectangle contains the Main()method and instantiates the Rectangle class.

C# Keywords

Keywords are reserved words predefined to the C# compiler. These keywords cannot be used as identifiers.
However, if you want to use these keywords as identifiers, you may prefix the keyword with the @ character.
In C#, some identifiers have special meaning in context of code, such as get and set are called contextual keywords.
