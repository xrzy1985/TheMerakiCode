Javascript | The things that can and will happen



Events

Javascript will react to certain events that are triggered during the execution of the HTML page. The HTML event can be something that a user intends to do, does, or something the browser does. A few examples of Javascript events are when HTML web pages finish loading, HTML input fields are changed, or an HTML button is clicked on. The events are there to signal the use of some kind of task. Javascript allows a programmer to execute code based on when events are detected. HTML allows event handler attributes to be added to HTML elements with accompanied Javascript code either linked to or within script elements. Let us take a look at the creation of a button within HTML, and we will add some Javascript code to it.

<!DOCTYPE html>
<html>
    <body>
    
        <p id="tutorial"></p>

        <button onclick="document.getElementById('tutorial').innerHTML=Date()">Time</button>

    </body>
</html>



You can use single or double quotation marks, but you must remember the order in which you use them. The code above illustrates how to change the content within the element using the id, tutorial. The time is displayed using a button, the button triggers an event, and the event is handled using our Javascript function Date(). We will see now how to change the content of an element with content already present.

<!DOCTYPE html>
<html>
  <body>

      <button onclick="this.innerHTML=Date()">Time</button>

  </body>
</html>



The code above is pretty simple. A simple HTML document with HTML, body, and button elements. The button has some text present before clicking the button. After the button is pushed, we change the text on the button, and we can do this with any element we want to. I just picked the button because I was lazy. The next example will show how we can change the text inside an element that has content already.

<!DOCTYPE html>
<html>
    <body>

        <button onclick="displayDate()">Time</button>

        <script>
            function displayDate() {
                document.getElementById("tutorial").innerHTML = Date();
            }
        </script>

        <p id="tutorial">This text will be changed once you click the button</p>

    </body>
</html>



HTML DOM events

HTML DOM events allow Javascript to register different event handlers on elements in any HTML document. Events are commonly used in conjunction with functions. The function will not be executed before the event has been triggered. The event could be a button click, mouse click, or a mouse hover.

What can Javascript Do?

Javascript built in event handlers can be used to handle and verify user input, verify user actions, and deal with all browser actions. The following is a list of DOM Events available inside Javascript:
