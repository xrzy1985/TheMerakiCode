Decision Making is an incredibly important aspect of programming with Python. Decision making is when the programmer understands that there might be more than one outcome from a certain situation. Those outcomes will drive the Python program in multiple different way according to the way the program has been written. The programmer must anticipate the way the specific conditions will occur and execute.
An if statement will essentially evaluate as True or False, and depending on how you write the if statement, it will either roll through the statement executing the code you give it or kick out of the block. Python does assume that any non zero / non null values are True, and if it is zero / null then it will evaluate as False.






If, If then else, Nested If Statements

Python provides three varieties of if statements

  • If statements
    • -- consisting of a boolean expression with one or more statements following
  • If then else statements
    • -- if statement that has the optional else statement that will execute when the if statement returns False
  • Nested If Statements
    • -- if statements that are inside a pre-existing if statement
