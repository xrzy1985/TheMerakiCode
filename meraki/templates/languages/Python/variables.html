Variables are locations in memory reserved by Python to store values being assigned. So, when a user creates a variable, x = "james", the user is really reserving space in memory for the string variable named x. Python will determine the best possible data type suited for the user once a variable is created. A variable can be a string, int, float, decimal, and many more data types. Python does not explicitly declare a data type when reserving memory for a variable, the declaration will occur after the first values are entered into the variable.
  • name_of_variable_int = 100
  • name_of_variable_float = 100.009
  • name_of_variable_string = "One Hundred
  • a = b = 3
    • This is a multi-assigment. This will assign bother a and b to the value 3.
  • a, b, c = 1, "JP", 31
    • This how to assign multiple values to multiple variables using the abstraction technique.
  • x = "This is how to " + str(1) + "add a string casted int into a sentence"
    • That will return the string typed in, but with the int variable returned as a string
