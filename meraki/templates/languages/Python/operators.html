Python's Assignment Operators

Python has assignment operators, which you should all be familar with by now.

The = Operator

Assigns the values from the right side operands to left side

The += Operator

It adds the right side to the left and assigns the result to the left operand

The -= Operator

Will subtrace the right from the left and assign the result to the left operand

The *= Operator

Will multiply the right with the left and assign the result to the left operand

The /= Operator

Will divide the right from the left and assign the results to the left operand

The %= Operator

Will perform left side % right side and assign the results to the left operand

The **= Operator

will take the the left side and raise it to the power of the right side and assign results to left operand



Python's Logical Operators

A Logical operator is when you have two variables and both conditionally equal one thing or the other. Let us assume that x = True and y = False

Logical AND operator

If both x AND y are true, then the condition evaluates as True

Logical OR operator

If either x or y are true, then the condition evaluates as True

Logical NOT operator

Used to reverse the state of the operand
Not(x and y) will evaluate to True

These could come in handy if you are evaluating two conditions based on certain circumstances

I can see this being useful for a traffic light system, when a detector needs to register more than one value and return certain states upon certain values.

Python's Identity Operators

Python has identity operators built into it. An identity operator will compare the memory locations of two objects.

When you have two objects and need to be sure if they are the same or not, use a identity operator

  • is operator
    -- Evaluates to true if the variables on either side point to the same object in memory
    -- x is y; if function(x) == function(y)

  • is not operator
    -- Evaluates to false if the variables on either side point to the same object in memory
    -- x is not y; if function(x) != function(y)
