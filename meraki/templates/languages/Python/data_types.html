Python Numbers

Python has three different number data types available to be used. Integer, Float, and Complex data types are the three available to Python users.



Integers

An integer can be created easily using Python. The integer value is the default data type for numbers used in Python. Integer values are usually whole positive or negative numbers, and d not have any decimal places. They are strictly real positive/negative whole numbers.

  • number = 100
  • num = number * 2
  • new_number = (num + 17) * 7
  • multiply_all = lambda a, b, c: a * b * c
  • print(multiply_all(number, num, new_number))

    A list of some acceptable integer values according to Python

  • 10
  • 1000
  • -78
  • -0x265
  • 0x96



Float values

A float value in Python is any number with or without a decimal place. A whole number can definitely be a float value, but Python will most likely cast the value as an int data type instead of float unless the user specifies otherwise. A float value stands for floating point real values.

    A list of some acceptable floating point values according to Python

  • 10.08
  • 32.4+e29
  • -78.000000000000002
  • -32.56e100
  • 70.1-E15



Complex Numbers

Python is fully equiped to handle complex numbers, like 3.14j or 45.J, but the complex number consists of the ordered pair of float numbers denoted by x and the imaginary unit denoted by j. Complex numbers are used in scientific programming, which some people steer clear of Python, but a lot of computer scientists use Python for complex scientific programming. Python can handle multithreading with the best of them, and the ability to be compiled allows Python to be a contender for running large calculations used by scholars.
