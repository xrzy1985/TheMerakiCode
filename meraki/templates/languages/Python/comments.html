Comments play a huge role in any programmers life. Comments can make or break a good team of programmers or just one. Comments allow the programmer to go back and review what has been coded, without needing to figure out exactly what the code does once again. A well placed comment can help other programmers figure out what you were doing and your specific thought process. Python has a variety of different comments we will discuss here.

Comments in Python

A comment can be made in three different ways in Python. The easiest comment to use and understand is the hash sign, #. From the # to the end of the line will be denoted as a comment, and Python will ignore the line during interpretation.





Comments cont...

A comment can be made using ''' , but you must remember to close off the comments using another ''' at the end. These types of comments are reserved for long comments, and they are particularly useful when needing to comment out an entire block or blocks of code quickly and easily.





Comments cont 2...

In conclusion, a comment can be created using a hash sign for single line comments, or use ''' to create a block comment. You must remember to close off the block comment or the entire py file will be commented out.
