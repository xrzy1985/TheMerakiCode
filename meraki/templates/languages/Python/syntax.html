Quotation within Python

Python will accept ", """, ', or ''' quotes to prepare the interpreter for a string literal. The string literal must start and end with the same type of quotation mark it began with. The ' and " are used within python to denote a string literal that is typically one line long. The ''' and """ are used when a string needs to be stretched across multiple lines.





Blank Lines

Blank lines are just white space among the python code. The blank lines could contain comments, which we will discuss next, but mostly they contain absolutely nothing. Python will completely ignore any white space that is not meant to be there already.



Keeping the Python file running
